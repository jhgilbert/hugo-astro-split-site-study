# Alerts — Hugo

All alert variants rendered with the Hugo alert shortcode.

## Info

{{< alert type="info" >}}
This is an informational alert. It provides helpful context.
{{< /alert >}}

## Warning

{{< alert type="warning" >}}
This is a warning alert. Something might need attention.
{{< /alert >}}

## Error

{{< alert type="error" >}}
This is an error alert. Something went wrong.
{{< /alert >}}

## Success

{{< alert type="success" >}}
The operation completed successfully.
{{< /alert >}}
