# User Stories

This document links to individual user story demonstrations for the Hugo/Astro split site study. Each story includes Playwright screenshots that are regenerated automatically when tests run.

## Stories

- [Seamless Navigation](user_stories/seamless-navigation.md) — The user can seamlessly navigate between Hugo and Astro pages
- [Tabs](user_stories/tabs.md) — The user can switch tabs with the tab nav
- [Alerts](user_stories/alerts.md) — The user can see contextual alert messages
- [Code Blocks](user_stories/code-blocks.md) — The user can view syntax-highlighted code
- [Collapsible Sections](user_stories/collapsible.md) — The user can expand and collapse content sections
- [Settings Toggles](user_stories/settings-toggles.md) — The user can toggle dark mode and whitespace density
